WHAT IT IS

DeChat is an ephemeral chat network. A router assigns you to the closest node by latency. Rooms are hosted on nodes and automatically expire after inactivity.

No identity. No analytics. No stored chat history.

HOW IT WORKS

Clients test node latency, request allocation from the router, then connect to the chosen node. Rooms can be sticky (mapped to the same node) to keep conversations consistent.

Designed for Raspberry Pi nodes and VPS routers.

PRIVACY

Messages live only in memory and disappear after inactivity. No logs by default.

STATUS

Project is in active development with a public network health dashboard.

OPEN STATS

Global network performance stats are publicly accessible and refresh automatically.

NODE / ROUTER OPERATOR REWARDS

We plan to incentivise infrastructure by rewarding node and router operators with a cryptocurrency token for keeping devices online and helping the network stay fast and reliable.

Rewards are intended to reflect uptime, stability, and real network contribution (details coming soon).
